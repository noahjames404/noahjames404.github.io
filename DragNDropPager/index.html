DragNDropPager

Customizable buttons that can be drag from one layout to another. 
An open source library for Android Developers.

Learn more 

Installation

Add it in your root build.gradle at the end of repositories:

allprojects {
  repositories {
    ...
	maven { url 'https://jitpack.io' }
  }
}

Add the dependency

dependencies {
   implementation 'com.github.noahjames404:dragndroppager:1.5.1'
}

Get the latest library here

Usage

  • Drag & Drop - buttons can be drag and drop, and migrate from one layout to another.
  • Customize Properties - buttons can be customize such as backgrounds.
  • Grid System - can be use to align buttons and size objects within the given format.
  • Supports ViewPager

Compatibility

  • Minimum SDK Version 16
  • Target SDK Version 29
