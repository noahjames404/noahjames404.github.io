Multiple Pages

A multiple page consist of a group Relative Layouts. Inside the xml type the following.

<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    tools:context=".MainActivity">

    <RelativeLayout
        android:id="@+id/page_one"
        android:layout_weight="1"
        android:layout_width="match_parent"
        android:layout_height="0dp"></RelativeLayout>
    <RelativeLayout
        android:id="@+id/page_two"
        android:layout_weight="1"
        android:layout_width="match_parent"
        android:layout_height="0dp"></RelativeLayout>
    <RelativeLayout
        android:id="@+id/page_three"
        android:layout_weight="1"
        android:layout_width="match_parent"
        android:layout_height="0dp"></RelativeLayout>

</LinearLayout>

Create 3 instances DNDPager and add the three layout's to its corresponding parameter, ensure that they have the same group_id e.i "mygroup".

RelativeLayout page_one = findViewById(R.id.page_one);
RelativeLayout page_two = findViewById(R.id.page_two);
RelativeLayout page_three = findViewById(R.id.page_three);

DNDPager pager1 = new DNDPager(page_one,3,3,"mygroup",this);
DNDPager pager2 = new DNDPager(page_two,3,3,"mygroup",this);
DNDPager pager3 = new DNDPager(page_three,5,5,"mygroup",this);

set setIsEditable() method to true to enable drag & drop feature to each DNDPager class

pager1.setIsEditable(true);
pager2.setIsEditable(true);
pager3.setIsEditable(true);

Create a list of DNDItems for each DNDPager.

List<DNDItem> pager1_items = new ArrayList<>();
pager1_items.add(new DNDItem("I'm alone",null, Color.GREEN,""));

List<DNDItem> pager2_items = new ArrayList<>();
pager2_items.add(new DNDItem("We",null, Color.GREEN,""));
pager2_items.add(new DNDItem("Happy",null, Color.WHITE,""));
pager2_items.add(new DNDItem("Few",null, Color.RED,""));
        
List<DNDItem> pager3_items = new ArrayList<>();
pager3_items.add(new DNDItem("What",null, Color.GREEN,""));

Then apply the corresponding list of DNDItems upon rendering each DNDPager. Try to run the code to check if its working.

pager1.render(pager1_items);
pager2.render(pager2_items);
pager3.render(pager3_items);
