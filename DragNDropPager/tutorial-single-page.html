Single Page

A single page only consists of a Relative Layout. Create a RelativeLayout with an id of single_page to an XML file.

<RelativeLayout
        android:id="@+id/single_page"
        android:layout_width="match_parent"
        android:layout_height="match_parent">
</RelativeLayout>

Create an instance of DNDPager and add the single_page to its constructor and set the number of rows & columns to 4.

RelativeLayout single_page = findViewById(R.id.single_page);
DNDPager pager = new DNDPager(single_page,4,4,"mygroup",this);

To enable the drag & drop feature.

pager.setIsEditable(true);

Create a list of DNDItems

List>DNDItem< items = new ArrayList<>();
items.add(new DNDItem("Drag Me",null, Color.GREEN,"tag1"));
items.add(new DNDItem("Drag Me too",null, Color.WHITE,"tag2"));
items.add(new DNDItem("No, Drag me First",null, Color.RED,"tag3"));

To view the items inside the page call the render() method and add it to it's parameter. Run the code to check if it is running.

pager.render(items);

Adding OnClick Listeners

The callback is applied to all buttons, a tag is used to distinguish each button. Ensure that setIsEditable() is set to false or it won't work.

pager.setOnClickBtnListener(new View.OnClickListener() {
            @Override
            public void onClick(View view) {
                switch (view.getTag().toString()){
                    case "tag1":
                        Toast.makeText(getApplicationContext(),"Wow",Toast.LENGTH_SHORT).show();
                        break;
                    case "tag2":
                        Toast.makeText(getApplicationContext(),"Hello",Toast.LENGTH_SHORT).show();
                        break;
                    case "tag3":
                        Toast.makeText(getApplicationContext(),"Bye",Toast.LENGTH_SHORT).show();
                        break;
                }
            }
        });

Update Page

updateButtons(List<DNDItem> items)  updates the contents inside the page. Contents can also be replace by another list of DNDItems by creating a new List and call the updateButtons() method.

items.get(0).text = "I've change";
pager.updateButtons(items);

Button Editor

While on edit mode users can customize the button's properties by double clicking it.

pager.setOnCustomize(new IDNDPager.ItemView() {
            @Override
            public void onCustomize(DNDPager pager, View view) {
                //your code
            }
        });
